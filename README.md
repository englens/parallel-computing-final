# parallel-computing-final
### Final project for CS 6068 - Parallel Computing.
Build/Run instructions:
for gpu (main) version, run 
```
./clean.sh
./build.sh
./run.sh
```
for the cpu version, run 
```
./clean_cpu.sh
./build_cpu.sh
./run_cpu.sh
```
This project must be run on linux, with CUDA installed.
